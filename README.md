# posts.codebase

Source code for practical examples from [flaviomilan.dev](https://www.flaviomilan.dev) blog posts.

Each directory corresponds to a blog post. Inside you'll find:

- `README.md` with a link to the original article, prerequisites, and setup instructions
- Ready-to-run code files
- `requirements.txt` with the required dependencies

## How to use

```bash
git clone https://github.com/flaviomilan/posts.codebase.git
cd posts.codebase

# navigate to the post you're interested in
cd langgraph-crewai-agno-getting-started

# follow the directory's README.md
```
